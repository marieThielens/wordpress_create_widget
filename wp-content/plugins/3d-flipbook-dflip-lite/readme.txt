== Installation ==

* Install dFlip lite Plugin by installing from your Wordpress Admin area
* Activate the module via the Plugins page in your Wordpress Admin area
* Use a shortcode to embed:

`[dflip source="example.com/wp-content/uploads/file.pdf"]`

* **source** is the product folder name under *wp-content/uploads*. (this is generated for you)


== Frequently Asked Questions ==

= How to create a flipbook from pdf =

First upload a pdf file to your wordpress site. Then, in the page/post write this code
`[dflip source="Path\To\PdfFile.pdf"]`
